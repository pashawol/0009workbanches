Стеллажная система

Стеллажная система
Предназначена для организации рабочего пространства и эргономичный системы хранения элементов бытового типа таких как: садовый инструмент, промышленный инструмент, коробки и канистры, разнообразных предметов личного использования.
Тип системы: наборная (размер системы хранения варьируется в зависимости от ваших нужд) межосевое расстояние при монтаже 816мм.

Комплектация

Направляющая двухрядная вертикальная

Используется как вертикальная стойка при монтаже систем хранения. Два ряда пазов предназначены для установки кронштейнов,экранов и полок.

  • Основной материал: сталь
  • Вес: 3.5 кг.
  • Габариты ВхШхГ: 2500х45х40 мм
  • Тип крепления: шурупами к стене

Направляющая горизонтальная Используется как горизонтальная закладная для фиксации двухрядных вертикальных направляющих.

  • Основной материал: сталь
  • Вес: 3 кг.
  • Габариты ШхГ: 54.5х32 мм.
  • (длина рассчитывается по согласованию с заказчиком)
  • Тип крепления: шурупами к стене

Перфорированный экран Используется для навешивания аксессуаров хранения (А1120; А1220; А1320; А1420; А1520; А1620; А1720; А1820; А2020; А2120).

  • Основной материал: сталь
  • Вес: 5 кг.
  • Габариты (ВхШхГ), мм - 550х800х15
  • Тип крепления: навешивание зип

Полка Используется для хранения габаритных элементов (коробки, канистры, колёса, садовый инструмент и т.д)

  • Основной материал: сталь
  • Вес: 3 кг.
  • Габариты (ВхШхГ), мм - 87х800х280
  • Тип крепления: навешивание зип
